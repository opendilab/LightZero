Welcome to LightZero’s Documentation

Overview

中文文档

LightZero is a lightweight, efficient, and easy-to-understand open-source algorithm toolkit that combines Monte Carlo Tree Search (MCTS) and Deep Reinforcement Learning (RL). You can find the source code on GitHub.

Related research papers include:

  • UniZero
