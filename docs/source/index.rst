Welcome to LightZero's Documentation
=====================================

Overview
--------

`中文文档 <./index_zh.html>`_

`LightZero <https://arxiv.org/pdf/2310.08348.pdf>`_ is a lightweight, efficient, and easy-to-understand open-source algorithm toolkit that combines Monte Carlo Tree Search (MCTS) and Deep Reinforcement Learning (RL). You can find the source code on `GitHub <https://github.com/opendilab/LightZero>`_.

Related research papers include:

- `LightZero <https://arxiv.org/pdf/2310.08348.pdf>`_
- `UniZero <https://arxiv.org/abs/2406.10667>`_
- `ReZero <https://arxiv.org/abs/2404.16364>`_
