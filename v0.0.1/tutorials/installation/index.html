Installation¶

LightZero is currently hosted on PyPI. It required python >= 3.7.

You can simply install LightZero from PyPI with the following command:

pip install LightZero

You can also install with the newest version through GitHub:

pip install -U git+https://github.com/opendilab/LightZero.git@main

In newest version of LightZero, cli is supported to do some data processing. Here is the version and help display.

LightZero is still under development, you can also check out the documents in stable version through https://opendilab.github.io/LightZero/.
